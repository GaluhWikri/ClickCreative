‚ú® Fitur Utama
Proyek ini mencakup beberapa komponen dan fitur interaktif:

Efek Parallax di Hero Section: Gambar latar belakang bergerak lebih lambat saat scrolling untuk menciptakan kedalaman visual yang imersif.

Header Adaptif (Sticky & Shrink): Bilah navigasi (header) menyusut dan menghapus bilah info atas (top bar) ketika pengguna mulai scroll.

Koleksi Produk Interaktif: Menampilkan empat kategori utama (Livestream, Sound System, Lighting, Accessories) dengan efek hover yang halus.

Custom Cursor: Menggantikan kursor default browser dengan kursor kustom yang membesar ketika diarahkan ke elemen yang dapat diklik (pointer).

Animasi Scroll (Intersection Observer): Menggunakan useScrollAnimation untuk menganimasikan kemunculan setiap bagian (<section>) saat masuk ke dalam viewport (fade-in-up).

Statistik dan Klien: Bagian khusus untuk menampilkan metrik kunci (4+ Tahun Pengalaman, 100% Klien Puas) dan daftar mitra klien.

Smooth Scrolling: Navigasi antar-bagian menggunakan tautan #hash diimplementasikan dengan fitur smooth scroll.
